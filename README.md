#ONE「一个」 Alfred Workflow

###1.安装

* bin 目录为 workflow 包，双击打开即可安装
* src 目录为源码

###2.使用

* 默认使用 `one` 拉取最新内容列表，可选择短篇，连载或问答
* 回车打开指定内容页面
